# Credit Card Analytics Dashboard (Power BI Project)

## Project Overview
This project analyzes credit card transaction and customer data to generate business insights using Power BI. The dashboard helps stakeholders understand revenue trends, customer behavior, and transaction patterns.

## Objectives
The main objective of this project is to build an interactive Power BI dashboard that provides insights into:

• Revenue trends
• Customer demographics
• Transaction behavior
• Card category performance
• Spending patterns

This helps businesses make data-driven decisions and monitor credit card performance.

## Tools & Technologies Used
- Power BI
- SQL
- Excel
- Data Modeling
- DAX

## Key KPIs
- Total Revenue: 55.32M
- Total Interest: 8M
- Total Transaction Amount: 45M
- Total Transaction Count: 656K

## Dashboard Features

### Transaction Analysis
- Revenue by Quarter
- Revenue by Card Category
- Revenue by Expenditure Type
- Revenue by Customer Job
- Revenue by Chip Usage

### Customer Analysis
- Revenue by Gender
- Revenue by Age Group
- Revenue by Salary Group
- Revenue by Marital Status
- Revenue by Education Level
- Top Contributing States

## Key Insights
- Blue credit cards generate the highest revenue.
- Swipe transactions are the most commonly used payment method.
- Customers aged 40–50 contribute the highest revenue.
- Male customers generate slightly higher revenue than female customers.
- Business professionals and high-income customers contribute significantly to revenue.
